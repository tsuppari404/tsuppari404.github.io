对汇编的小理解

用c++的方式来理解

mov ax,001AH -> ax=001A (H表示16进制，001A是一位16进制数)
add ax,85H -> ax+=85
add ax,bx -> ax+=bx

进行数据传送或运算的时候，指令的两个操作对象的位数应当是一致的，以下为错误指令

mov ax,bl (bl为8位寄存器，ax为16位寄存器)
mov al,20000 (al为8位寄存器，数据最大为2^8=256，0-255，而ax为16位寄存器，数据最大为2^16=65536,0-65535，均为十进制)
mov al,100H (100H已超出，16进制中1位相当于2进制中4位)

注意al和ah是构成ax的两个8位寄存器，我第一次看的时候没注意蒙了半天

之前第一张图的代码用C++理解，代码如下(因为我对C++更熟悉)

  1. ax=001AH;
  2. bx=0026H;
  3. al+=bl;

    以下省略…推荐在草稿纸上自行演示

    计算机应用课上自己打的代码…因为实在是…懂得都懂，输入两个十六进制数，输出它们相加的结果(注意是大写字母)

输入样例：
4E20
406
输出样例:
5226

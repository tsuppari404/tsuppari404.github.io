忙碌的10月

Posted on 2021-11-06
  • T5uppari
Home

终于过完了一个忙碌的 10 月份，比赛都集中在 22 号 - 24 号这个周末，这三天礼拜五到礼拜天，一共打了 4 场比赛，江西高校网络安全大赛线下赛、icpc、柏鹭杯、江西省大学生信息安全技术大赛，这里两场省赛一场是东华理工主办的一场是我们自己学校主办的。好在最后结果还能接受。

# 10.22 东华理工的线下赛

第二次打线下赛，上午是 ctf 下午是 awd。

上午 ctf 是开局并不算顺利，我上场🐏了一个 RSA 的签到题拿了个一血，然后剩下 3 个多小时都在罚坐，那题直接 yafu 分解 n 没什么说的。然后第二题现在看来也不算难，在风信子的师傅点拨之后也是复现出来了。在 crypto 的挨打之路记录了详细题解。风信子太强了呜呜呜。江西理工的师傅上来把 web 题🐏光了，太强了。最后上午 ctf 打了个第五名。

下午 awd，老实说我一个密码手确实是抱着摆烂的心态出去的，想着帮他们弄下源码写下自动交 flag 的脚本 (后面也没写出来呜呜呜因为根本没有 flag)，下午 awd 真的是超级憋屈。pwn 好像全场都没出题，基本上都是 web 在打，我们队感觉没干什么都宕机了几次，三次重置机会很快就用完了，好像是隔壁江软的师傅们杀人还诛心。

总之就是被锤烂了，用主办方给的一个洞上了一点分，最后 awd 第四名，团体打了个第五名拿了个三等奖，有个奖金十佰，也是第一次打比赛拿的奖金。

日常摆烂～

晚上一顿胡吃海喝滚回学校了，第二天还有比赛，难顶...

# 10.23icpc

这场比赛一开始是没想要报名的，指导员前一个礼拜拉咱们说这个比赛获奖率有 60%，让我们冲一冲。

然后早上 7 点钟屁颠屁颠地爬起床去实验室打线上了，麻了这个摄像头，前面后面旁边到处都拍着。高中打过 noip，带了两个翻译就来比赛了。上来咔咔三题到了 90 多名，后面也一直没做出题来了。唉，要不是有一题有一次提交错误的记录被罚了 20 分钟，这次比赛是双赛道，一套题两个奖，打一场比赛有两个奖来分，可惜了不然还能拿一个教育局的省三等奖。

最终排名是这样

日常摆烂！

最后只拿了一个 icpc 的铜奖，我感觉如果是这种难度，我高中的学弟上来能乱杀直接抱个金奖走。

感谢山东建筑大学曹哥的指导（磕头 ing）

# 10.24 柏鹭杯 & 自家办的省赛

又是早上屁颠屁颠爬起来，柏鹭杯竞争还是挺强的，看着报名的队伍大多都是 985 和 211 的，还有好多警院，瑟瑟发抖...

柏鹭杯上来也是🐏了密码的第一题，大数据分解，真就硬分解，yafu 跑了 10 分钟吧把 n 分解出来了，当时没抱啥希望就后台在那挂着，然后看了一眼居然出来了，当时那叫一个激动，赶紧把 flag 交了，也算是我们队的第一炮。然后两题密码，第二题有点脑洞，说起来也简单

第三题就是原题了，第一次在这种大比赛把密码 ak 了。一队的师傅们太厉害了，一队拿了三等奖打了 1 个 w，xmsl，我们打了十多名摸了个优胜奖走。

晚上还有时间打了下自家办的省赛的初赛，u1s1 真的很想喷火种这个平台，团体赛硬是打成了个人赛，选择题每个队员都要提交一遍，有点阴间。最后一小时一队的师傅们咔咔交 flag 把我们给🐏了，呜呜呜，他们拿了个院里面的一等奖。

# 10.30 自家办的省赛 - 线下赛

第 3 次打线下的 awd，居然是在自己学校打。说实话感觉很草率，没有主持，没有颁奖，外面的师傅过来打完就走了，连一个颁奖仪式都没有，因为有和东华理工的对比，唉还需要多进步吧。

这次 awd 倒是体验感很好，干起了 web 的事情，嘿嘿 web 真好玩，上来把源码下下来，然后 D 盾扫，哇靠一堆洞，补！打！

拿了一堆 flag 嘿嘿嘿，我们队的防护做的也还行，没怎么被打，就是没写出自动化交 flag 的脚本，一直在手动盖章，太菜了呜呜呜

最后结果也还行，也不知道是什么奖。

三人摆烂合照～

感谢宋师父一直以来的指点（磕头 ing），挂一个宋师父的博客嘿嘿嘿
